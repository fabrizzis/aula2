1) O que é um repositório?
É um diretório onde os arquivos do seu projeto ficam armazenados, podendo armazenar
código, imagens, audios ou qualquer coisa relacionada ao projeto.

2) Qual a diferença entre Git e Github
O GIT é um sistema de controle de código, sendo gratuito. O GITHUB é uma plataforma de 
desenvolvimento em que você pode armazenar códigos, gerenciar seu projetos e construir 
softwares e aplicações.

3) Porque utilizar controle de versão para os arquivos de um projeto de desenvolvimento de software?
Em principal ele é usado para permitir que exista o trabalho em equipe. Com ele, todas as pessoas tem acesso aos arquivos podendo
observar todas as outras mudanças que ocorreram.


4) Quais vantagens você enxerga no Git? Você utilizaria em seu projeto?
Vantagens: 
Facilidade de trabalhar;
Capacidade de mover e adicionar arquivos;
Verificar status das modificações;

A principio, não pensamos em usar o GIT em nosso projeto. Mas vejo que é uma boa opção usa-lo, ainda mais quando tivermos mais 
conhecimento.
